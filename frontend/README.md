# 前端

這個資料夾已初始化為 Vue 3 + Vite 前端專案。

## 安裝依賴

```powershell
npm install
```

## 啟動開發伺服器

```powershell
npm run dev
```

預期開發伺服器網址：

```text
http://localhost:5173
```

## 可用指令

- `npm run dev`：啟動開發伺服器
- `npm run build`：建立正式版檔案
- `npm run preview`：預覽正式版結果

## 後端 API

目前後端預設網址：

```text
http://localhost:8000
```
