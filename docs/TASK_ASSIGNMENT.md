# 團隊分工

## 成員角色

| 成員 | 負責範圍 |
| --- | --- |
| A | 核心負責人、專案結構、API 串接、Git 流程 |
| B | 前端頁面、使用者條件輸入 |
| C | 前端餐廳列表、餐廳詳細頁與收藏畫面 |
| D | 後端推薦 API、餐廳 API、收藏 API |
| E | 文件、mock data、測試、簡報資料 |

## 初始任務

| 任務 | 內容 |
| --- | --- |
| Task 1 | 建立專案結構與文件 |
| Task 2 | 建立餐廳列表與詳細頁前端 |
| Task 3 | 建立餐廳 API |
| Task 4 | 建立推薦 API |
| Task 5 | 建立收藏 API |
| Task 6 | 補充餐廳與菜單 mock data |
| Task 7 | 測試 API 並整理簡報資料 |
