# 專題規格

## 專案名稱

Mood Food Recommendation：心情導向餐廳與菜單推薦系統。

## 專案目標

使用者輸入心情、預算、人數、用餐目的、餐廳類型、城市與行政區後，系統會根據 mock 餐廳資料計算推薦分數，並依照分數由高到低回傳推薦餐廳與推薦理由。

## MVP 功能

1. 使用者條件輸入
2. 餐廳推薦計分
3. 餐廳詳細資料查詢
4. 餐廳菜單查詢
5. 收藏餐廳
6. 查看收藏清單

## Phase 2 功能

1. 使用者登入與個人化偏好
2. AI 產生更自然的推薦理由
3. 串接 Google Maps API 與真實距離查詢
4. 用餐紀錄與推薦回饋
5. 正式資料庫與後台管理工具

## MVP 不做的事

1. 不串接第三方地圖或外送平台 API
2. 不建立完整會員系統
3. 不建立後台管理頁
4. 不使用正式資料庫，先以 JSON mock data 完成流程
